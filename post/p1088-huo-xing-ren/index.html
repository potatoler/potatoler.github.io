C++有一个特别神奇的库叫做,算法，名字就说明其中注定是有一些神奇的函数。其中有一个很快能解决此题的函数：

next_permutation

这个函数每运行一次就可以排列出下一个全排列的序列，和sort()用法类似，具体信息戳上面函数名...

int tot=n;//所有全排列的数量
for(int i=0;i<tot;i++){
	next_permutation(a,a+n);
}

Ann=n!A^{n}_{n}=n!Ann​=n!

也就是tot的取值；

	for(int i=0;i<n;i++) scanf("%d",&a[i]);

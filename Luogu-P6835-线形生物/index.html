Luogu P6835 线形生物

2020-09-20 期望DP ， DP ， 题解

题目描述

线形生物要从 111 号台阶走到 n+1n+1n+1 号台阶。最开始，1,2,3,...,n1,2,3,...,n1,2,3,...,n 号台阶都有一条连向下一台阶的有向边 i→i+1i→i+1i→i+1，之后加入 mmm 条「返祖边」ui→vi(ui≥vi)u_i→v_i(u_i≥v_i)ui​→vi​(ui​≥vi​)，构成一个返祖图。线形生物每步会等概率地选取当前台阶的一条出边并走向对应的台阶，走到 n+1n+1n+1 号台阶便会停止行走。求线性生物总共走的步数的数学期望，答案对 998244353998244353998244353 取模。

思路一

设 fif_ifi​ 为从 iii 号台阶走到 i+1i+1i+1 号台阶的期望步数，当走到 iii 号台阶时，假设它有 kik_iki​ 条出边，那么便有 1ki\frac{1}{k_i}ki​1​ 的概率走向下一个台阶，有 ki−1k\frac{k_i-1}{k}kki​−1​ 的概率走返祖边回到前面的台阶。前者需要的步数代价为 111，后者需要的步数代价是走返祖边后到下一个台阶的期望步数。记 sumsumsum 为走每一条返祖边回到 iii 点的期望步数总和，具体来讲，记 EEE 为 iii 号台阶的返祖边集，有 sum=∑e(i,j)∈E(∑ji−1fj+1)sum=\sum_{e(i,j)\in E} (\sum_{j}^{i-1}f_j+1)sum=∑e(i,j)∈E​(∑ji−1​fj​+1)​，加一是因为返祖边本身需要一步。一共有 ki−1k_i-1ki​−1 条返祖边，每条边走的概率均等，走一次返祖边回到 iii 的期望步数就是 sumki−1\frac{sum}{k_i-1}ki​−1sum​。但是现在只回到了 iii ，要想走到下一个台阶，还需要加上从 iii 走到下一个台阶的期望步数，也就是 fif_ifi​。对刚才推出的式子进行化简：

fi=1k⋅1+k−1k(sumk−1+1)k fi=1+sum+(k−1)fifi=sum+1\begin{aligned} f_i&=\frac{1}{k}·1+\frac{k-1}{k}(\frac{sum}{k-1}+1) \\ k\ f_i&=1+sum+(k-1)f_i \\ f_i&=sum+1 \end{aligned} fi​k fi​fi​​=k1​⋅1+kk−1​(k−1sum​+1)=1+sum+(k−1)fi​=sum+1​

#include<iostream>
#include<cstdio>
#include<cstdlib>
#include<cmath>
#include<climits>
#include<algorithm>
#include<vector>
#define int long long
using namespace std;
const int MaxN = 1000006, Mod = 998244353;
int degree[MaxN], f[MaxN], sum[MaxN]; // 注意这里的sum和上面所说的sum意义不同
vector<int> to[MaxN];

inline int Read(){
	int num = 0, op = 1;
	char ch = getchar();
	while(!isdigit(ch)){
		if(ch == '-') op = -1;
		ch = getchar();
	}
	while(isdigit(ch)){
		num = num * 10 + ch - '0';
		ch = getchar();
	}
	return num * op;
}

inline int Get(int x){
	int ans = 0;
	for(int i=0; i<to[x].size(); i++){
		if(to[x][i] == x) ans = (ans + 1) % Mod;
		else ans = (ans + sum[x-1] - sum[to[x][i]-1] + 1 + Mod) % Mod;
	}
	return ans;
}

signed main(){
	int id = Read(), n = Read(), m = Read();
	for(int i=1; i<=m; i++){
		int u = Read(), v = Read();
		degree[u]++;
		to[u].push_back(v);
	}
	for(int i=1; i<=n; i++){
		f[i] = (Get(i) + 1) % Mod;
		sum[i] = (sum[i-1] + f[i]) % Mod;
	}
	printf("%lld", sum[n]);
	return 0;
}
